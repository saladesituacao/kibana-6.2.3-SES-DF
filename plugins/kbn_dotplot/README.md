# Dotplot for Kibana

This is a plugin developed for Kibana 5 that you can build dotplot vis


## Installation Steps

```
cd KIBANA_HOME/plugins
git clone https://github.com/dlumbrer/kbn_dotplot
cd kbn_dotplot
npm install
*Run your Kibana*
```
> **Important:** If you have any problem with the plugin version (like a warning message "**it expected Kibana version "5.4.Â·", and found "5.4.X"**") only change the value of the "version" tag on the package.json to your Kibana version


#### Uninstall:
```
cd KIBANA_HOME
rm -rf plugins/kbn_dotplot/
```
